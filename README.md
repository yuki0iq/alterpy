# [alterpy](https://yuki0iq.github.io/alterpy)

First launch:

```sh
python3 -m venv venv
source venv/bin/activate

# TODO replace with init
python3 -m pip install -r requirements.txt
# <- config files
python3 -m alterpy.init
```

Start:

```
source venv/bin/activate
python3 -m alterpy
```
